# scoverplots
Reproduce plots for scover manuscript. Output will be generated in `output` dir. 

## Kidney

First, un-tar the kidney_data.tar.gz in the directory `data`:

```
cd data
tar -xzvf kidney_data.tar.gz
cd ..
```

Then, run the script:

```
Rscript kidney.R
```
